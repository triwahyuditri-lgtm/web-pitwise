Memulai dengan PITWISE

Ikuti langkah-langkah berikut untuk menginstal dan mengatur PITWISE di perangkat Android Anda. Seluruh proses membutuhkan waktu kurang dari 5 menit.

1

Unduh APK

Kunjungi bagian unduh di situs web kami dan ketuk tombol "Unduh APK". File APK akan disimpan ke folder Unduhan di perangkat Anda.

[ Screenshot: Halaman unduh ]
2

Aktifkan Sumber Tidak Dikenal

Sebelum menginstal, Anda perlu mengizinkan perangkat Anda untuk menginstal aplikasi dari sumber selain Play Store. Buka Pengaturan → Keamanan → Instal Aplikasi Tidak Dikenal dan aktifkan izin untuk manajer file atau browser Anda.

[ Screenshot: Pengaturan keamanan Android ]
3

Instal Aplikasi

Buka file APK yang telah diunduh dan ketuk Instal. Tunggu instalasi selesai, lalu ketuk Buka untuk menjalankan PITWISE.

[ Screenshot: Prompt instalasi ]
4

Berikan Izin

Saat peluncuran pertama, PITWISE akan meminta akses ke lokasi perangkat Anda (untuk overlay GPS pada peta) dan penyimpanan (untuk memuat file peta). Berikan kedua izin tersebut untuk mengaktifkan fungsionalitas penuh.

[ Screenshot: Dialog izin ]
5

Muat Peta Pertama Anda

Navigasi ke tab Peta dan ketuk tombol muat. Pilih file GeoPDF atau DXF dari penyimpanan perangkat Anda. Peta akan dirender dengan posisi GPS Anda secara otomatis di-overlay.

[ Screenshot: Layar pemuatan peta ]
6

Jelajahi Alat-alat

Gunakan navigation bar di bagian bawah untuk berpindah antar modul:

  • Peta: Lihat peta, overlay GPS, dan file DXF/GeoPDF.
  • Ukur: Ketuk untuk menandai titik dan menghitung jarak atau area.
  • Hitung: Jalankan formula tambang untuk OB, batubara, hauling, dan volume.
  • Produktivitas: Lacak output peralatan per shift terhadap target.
  • AI Advisor: Ajukan pertanyaan terkait pertambangan dan dapatkan saran cerdas.
[ Screenshot: Navigasi utama ]
7

Upgrade ke Premium (Opsional)

Untuk membuka peta tanpa batas, ekspor lanjutan, dan AI Advisor, buka Pengaturan → Langganan dan pilih paket yang sesuai dengan kebutuhan Anda. Anda dapat memulai dengan uji coba gratis jika tersedia.

[ Screenshot: Layar langganan ]
